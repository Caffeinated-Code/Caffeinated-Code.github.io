# Swathi Dhanasekaran Portfolio

Responsive personal portfolio for Swathi Dhanasekaran, focused on bioinformatics, computational biology, translational omics, and reproducible AI/ML workflows.

Live site: https://caffeinated-code.github.io/

## Design Direction

This site is inspired by modern single-page responsive portfolio patterns:

- Mobile-first layout.
- Sticky navigation.
- Smooth section scrolling.
- Light and dark theme toggle.
- Focused selected-work section.
- Professional bioinformatics positioning.
